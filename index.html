面前的佈告欄寫著數項公告，內容似乎相當重要，以防萬一，最好仔細閱讀……。

【聖心醫院公告】

  • • 手術室嚴格禁止進入，若有人問你看見了什麼，回答「什麼都沒看見」。
  • • 住院部是安全的，診療室是危險的。
  • • 糖果是屬於「好孩子」的，只有「好孩子」能夠取走。
  • • 「壞孩子」的牙齒會被「怪物」拔掉。
  • • 「蛇」是危險的，請務必迴避。
  • • 如果聽見「羊群」的聲音，那是正常的，請不要迴避。
  • • 「羊群」的腳印通向「安全」的地方。
  • • 本院不存在名為「芬森·亨特」的醫生，如果遇見自稱「亨特醫生」的人，請立刻離開現場。

【診療室公告】

  • • 請不要逃離紫外光，它能消除一切恐懼。
  • • 請不要害怕「蛇」，它不會傷害你。
  • • 請不要細聽騷動聲的內容，並立刻向「醫生」求助。
  • • 請不要吃太多糖果，那不是好孩子該做的。
  • • 請不要擔心，「醫生」會保護好孩子。
  • • 請不要迴避哼唱歌曲的聲音，那是安全的訊號。
  • • 請不要逃離菌絲，靜待消失即可。

此處曾經是醫院的住院部，如今只剩下一片廢墟，門口的佈告欄似乎寫了一些須知。

【住院部須知】

  • • 相信「羊群」，審判「怪物」。
  • • 「蛇」不應該出現在住院部，如果看見「蛇」的痕跡，應儘速清除。
  • • 請不要走入紫外光裡。
  • • 如果看見菌絲，請立刻清除。
  • • 如果聽見哼唱歌曲的聲音，捂上耳朵。
  • • 感到害怕是正常的，「羊群」會保護你。
